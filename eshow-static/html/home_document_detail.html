Lor是一个运行在OpenResty上的基于Lua编写的Web框架，当前最新版本是0.0.2预览版，重构了之前发布的0.0.1版本。主要API特性已经freeze，下个版本为0.0.3，着重于修复当前路由的匹配bug。
特性
  1. 路由采用Sinatra风格，Sinatra是Ruby小而精的web框架.
  2. API基本采用了Express的思路和设计，Node.js跨界开发者可以很快上手.
  3. 支持插件(middleware)，路由可分组，路由匹配支持string/正则模式.
  4. lor以后会保持核心足够精简，扩展功能依赖middleware来实现. lor本身也是基于middleware来实现的.
  5. 推荐使用lor作为HTTP API Server，lor此后也会支持模板渲染/Session/Cookie等常规web功能.
  6. 提供了一个cli工具lord,可以一键生成项目骨架/启动、停止项目，快速体验OpenResty和lua开发.
